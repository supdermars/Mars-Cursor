## 安装说明

### 方法一：直接使用可执行文件（推荐）

1. 下载最新版本的 `Mars-Cursor.exe`（Windows）或 `Mars-Cursor.app`（macOS）
2. 确保 `images` 文件夹与可执行文件在同一目录下
3. 管理员运行程序

### 方法二：从源代码运行

1. 克隆或下载本项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行主程序：
   ```bash
   python main.py
   ```
